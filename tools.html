More Details

By default, the basic system installer is able to install apps.
Additionally, you can use the Device Portal to do that.

However, tools like the Astoria Package Manager can assist in retrieving packed apps from the repository.
It also helps you to install games along with their related data,
which is not possible with the normal installer.

USE ADB

To install APK apps directly from your PC, you need to use ADB (Android Debug Bridge).
However, ADB doesn't work directly with Astoria. First, you'll need to connect your phone using tool called wconnect.

Get WConnect & ADB [ Download]

Setting Up WConnect Agent

  1. Prepare a PIN code from Settings > Update > For Developers.
  2. At Device Discovery, enable the toggle.
  3. Click on Pair, then use the given pin in the command below.
  4. Open CMD on your PC.
  5. Navigate to the wconnect folder using: cd "Location of wconnect folder"
  6. Connect your phone using the command: wconnect usb <pin>
  7. Or via IP command: wconnect 198.168.0.xxx <pin>
  8. Once your phone is connected, you can start using ADB commands.
  9. ADB install command: adb install "Path to APK app"
