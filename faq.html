QUESTIONS & ANSWERS

Can I interop Unlock?

Yes, generally features cabs will make initial interop unlock.
you can later perform full interop unlock using Interop Tools

Interop Unlock Guide ( PDF |  YouTube)

For 950s go with MassStorage option, much easier

Can I boot to MassStorage?

Yes, but with the Developer menu, which will also be added with the features cabs
This build isn't supported by WPI, so you may not be able to switch to MassStorage with it.

Steps

Switch to MassStorage using Developer Menu

  • - Reboot your phone
  • - Then hold Volume Down to show Developer Menu
  • - Select MassStorage, press confirm button

Can I enable RootAccess?

Yes, but keep in mind:

This feature still experimental use it at your own responsibility


1st MassStorage

Switch to MassStorage using Developer Menu

2nd Enable RootAccess

  • - [ Download] WPinternals-2.9.2 (ProjectA).zip
  • - Start WPinternals (ProjectA)
  • - Go to Enable root access
  • - Click Unlock phone
  • - Done, root access enabled

You can revert this step anytime with the same steps but using Disable root access

Can I update to higher builds later?

Yes, using offline updater

but you will lose Astoria.
Astoria currently only works with builds 14320 and below.

Why should I change device time?

Those are test signed builds and not production signed builds.
You cannot update to them unless you have date before the default expiry date.

Do those builds expire later? No.

However, you can set the time back once you finished the update process

Why should I disable WiFi?

To avoid the time reset to the current time.

However, you can enable WiFi back once you finished the update process

What does each (Stage) do?

Stage 01

It will enable FlightSign, as mentioned above you need it because the build is test signed.

Stage 02

It will push 14320 update along with features cabs such as:
  • - ProjectA
  • - CMD Injector
  • - CPU-Z test app

Should I enable all (Features cabs)?

Yes, otherwise you will have difficulties to boot to MassStorage,
even to Interop unlock, since this build isn't officially supported by WPI.
