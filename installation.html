REQUIREMENTS

  • - Phone with W10M build 10586
  • - [Download] Astoria update package
  • - [Download] Universal Updater 2.7.1 (Or higher)
  • - [Download] Windows Phone Internals (WPI)

PRE-UPDATE

  • - Don't connect WiFi before you finish
  • - Battery charged 50% or more

UPDATE STEPS

PART 1

  1. Flash Retail FFU Skip if you're on 10586
  2. Set Phone date to 5th May 2016
  3. Unlock bootloader (PDF | YouTube)

PART 2

Push cabs below on stages not at once.

For easy setup use the latest Universal Updater 2.7.1+

  1. (Stage 1) Push FlightSign cab
  2. Wait until system update..

  3. (Stage 2) Push 14320 cabs
  4. Wait until system update..
  5. Done

UNIVERSAL UPDATER GUIDE

  1. Connect your device, ensure connection is well
  2. Open the app, choose (1. Device)
  3. Then choose Offline update packages
  4. Copy path of the target stage
  5. Paste the Offline packages path, press Enter
  6. Universal Updater will display features list
  7. Ensure required features are enabled
  8. Press '0' to Confirm
  9. Choose 1. Filter packages (Important to choose filter)
  10. Read packages details, ensure date set to (5th May 2016)
  11. Ensure WiFi is not connected
  12. If everything ready, choose 1. Push packages

Once iutool finished pushing packages, navigate to:

  1. Settings
  2. Update & Security
  3. Phone Update

And monitor the progress. If you received any error,
check the dialog that appear after push process.

Usually (Stage 01) is short.
But (Stage 02) will take time,
so please wait and don't interrupt the process.

POST UPDATE

The update will add test app (CPU-Z) at your files (Arcadia) folder.
You can just click on it and install to test if the update working.


Keep in mind first installation will take time until Astoria initialize


Once update completed you can set the date back and enable WiFi.

Troubleshoot

  • If there was no progress at Phone Update after pushing cabs, try again.
