 DEVICE UPDATE

 REQUIREMENTS

  • - Phone with W10M build 10586
  • - [ Download] Astoria update package
  • - [ Download] Universal Updater 2.7.6 (Or higher)
  • - [ Download] Windows Phone Internals (WPI)

 PRE-UPDATE

  • - Don't connect WiFi before you finish
  • - Battery charged 50% or more

 UPDATE STEPS

PART 1

  1. Flash Retail FFU Skip if you're on 10586
  2. Set Phone date to 5th May 2016
  3. Unlock bootloader ( PDF |  YouTube)

PART 2

Push cabs below on stages not at once.

For easy setup use the latest Universal Updater 2.7.6+

  1. (Stage 1) Push FlightSign cab
  2. Wait until system update..

  3. (Stage 2) Filter & Push 14320 cabs

    4. (Do not push all/multiple cabs without filtering them for your device,
    if you are not sure how to; check Universal updater below)

  4. Wait until system update..
  5. Done

 UNIVERSAL UPDATER GUIDE

1.Connect & Select Packages

  1. Connect your device, ensure connection is well
  2. Open the app, choose (1. Device)
  3. Then choose Offline update packages
  4. Universal Updater will ask to select packages folder
  5. Choose the Offline packages folder

2.Review Features

Required features enabled by default

  1. Universal Updater will display features list
  2. Ensure required features are enabled
  3. Press '0' to Confirm (If question appeared)

3.Filter & Push

By default (14320 Stages) will force filter mode

  1. Choose 1. Filter packages (If question appeared)
  2. Read packages details, ensure date set to (5th May 2016)
  3. Ensure WiFi is not connected
  4. If everything ready, choose 1. Push packages

4.Monitor Progress

Once iutool finished pushing packages, navigate to:

  1. Settings
  2. Update & Security
  3. Phone Update

And monitor the progress.
If you received any error, check errors list below.


Usually (Stage 01) is short.
But (Stage 02) will take time,
so please wait and don't interrupt the process.

 POST UPDATE

The update will add tools & test app (CPU-Z) at your files (Apps) folder.

  1. Apps\CMDInjector
  2. Apps\WUT Mini
  3. Apps\Arcadia\CPU-Z

To test Astoria, try to install CPU-Z
You can just click on it and install to test if the update working.

Keep in mind first installation will take time until Astoria initialize


Once update completed you can set the date back and enable WiFi.

 ROOT ACCESS (14320)

This feature still experimental use it at your own responsibility


1st MassStorage

Switch to MassStorage using Developer Menu

  • - Reboot your phone
  • - Then hold Volume Down to show Developer Menu
  • - Select MassStorage, press confirm button

2nd Enable RootAccess

  • - [ Download] WPinternals-2.9.2 (ProjectA).zip
  • - Start WPinternals (ProjectA)
  • - Go to Enable root access
  • - Click Unlock phone
  • - Done, root access enabled

 TROUBLESHOOT

  • If there was no progress at Phone Update after pushing cabs, re-open Phone Update menu after few seconds and check again. If nothing started, then try to push again.

Error Code Description
0x800b010a Signature Verification Issue
Solution: Enable flight signing
0x800b0101 Incorrect Date and Time
Solution: Change date
0x80188302 Package already present on the image
0x80188305 Duplicate packages found in update set
0x80188306 File Collision or Not Found
Solution: Ensure list matches InstalledPackages.csv
0x80188307 Package DSM or contents are invalid
Solution: Keep only one type in the folder (CBS or SPKG). Choose the correct package type
0x80188308 Not enough space
0x800b0114 Certificate has an invalid name
Solution: Enable flight signing
0x80004005 E_FAIL
Solution: Reboot the phone and try again
0x802A0006 Try with another PC
